# Commands

## Package Manager

Prefer using **pnpm**.
Fallback to npm if pnpm is not available.

---

## Install dependencies

```bash
pnpm install
```

or

```bash
npm install
```

---

## Run development server

```bash
pnpm dev
```

or

```bash
npm run dev
```

---

## Build project

```bash
pnpm build
```

or

```bash
npm run build
```

---

## Start production server

```bash
pnpm start
```

or

```bash
npm run start
```

---

## Lint code

```bash
pnpm lint
```

or

```bash
npm run lint
```

---

## Format code

```bash
pnpm format
```

or

```bash
npx prettier . --write
```

---

## Run tests

```bash
pnpm test
```

or

```bash
npm run test
```
